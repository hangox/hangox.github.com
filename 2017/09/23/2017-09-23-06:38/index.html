Metaweblog 的格式

Metaweblog的实现主要是参考这个开源项目,这个源码写得很直白，非常好懂。简单一点解释Metaweblog Api(后面简称M)，和M对接的url其实只有一个，然后把访问不同的方法名封装在一个XML里面，通过解析这个xml，我们就能够知道这个请求到底需要访问什么方法。这个和我们一般的Api都不太一样，我在这个地方迷茫了好久。知道这个之后剩下的就是xml解析的问题了
